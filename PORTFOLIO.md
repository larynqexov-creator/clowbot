# PORTFOLIO (template)

Правило: активных проектов одновременно 3–7. Остальные = PAUSED.

Columns:
- Project
- Area (Business/Science/Personal)
- MoneyPotential (0-10)
- Urgency (0-10)
- Leverage (0-10)
- StrategicValue (0-10)
- RiskPenalty (0-10)
- Score
- Status (ACTIVE/PAUSED/DONE)
- Next Action
- Owner
